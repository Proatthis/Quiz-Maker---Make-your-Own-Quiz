# QuizMaker

A browser-based quiz app built in C++ and compiled with Emscripten. Runs offline, loads instantly, and works on any device.

## Features
- Multiple-choice questions
- Instant feedback and scoring
- No internet required after loading
- Lightweight and fast

## How to Run
1. Open `quiz.html` in any browser
2. Answer the questions
3. View your final score

## How It Works
- Written in C++
- Compiled to WebAssembly using Emscripten
- Deployed via Vercel
